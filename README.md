# AI角色扮演平台

## 项目简介

AI角色扮演平台是一个基于PHP开发的Web应用程序，允许用户与各种精心设计的AI角色进行智能对话。平台提供了完整的用户系统、角色管理、聊天对话、积分系统、订阅功能以及强大的后台管理功能。

## 主要功能

### 用户端功能

#### 1. 用户系统
- **用户注册/登录**：支持邮箱注册，包含邮件验证码验证
- **个人中心**：用户信息管理、头像设置、密码修改
- **忘记密码**：通过邮箱验证码重置密码

#### 2. AI角色浏览与搜索
- **角色广场**：浏览所有公开的AI角色
- **分类筛选**：按角色分类快速筛选
- **智能搜索**：支持按角色名称或简介搜索
- **排序功能**：支持按受欢迎度、评分、创建时间、名称排序
- **角色详情**：查看角色介绍、评分、使用次数等信息

#### 3. 聊天对话系统
- **实时对话**：与AI角色进行实时对话交流
- **流式传输**：支持AI回复的流式输出，提供流畅的对话体验
- **会话管理**：支持创建多个对话会话，管理历史对话记录
- **角色切换**：在对话中快速切换不同的AI角色
- **消息管理**：支持删除、重新生成、继续生成消息
- **上下文记忆**：AI能够记住对话上下文（限制30条消息）

#### 4. 角色创建与管理
- **创建角色**：用户可以创建自己的专属AI角色
- **角色定制**：设置角色名称、头像、介绍、提示词（Prompt）
- **提示词优化**：ai智能优化提示词（Prompt）
- **我的角色**：管理自己创建的所有角色
- **角色发布**：选择角色是否公开给其他用户

#### 5. 订阅系统
- **角色订阅**：订阅感兴趣的AI角色，方便快速访问
- **订阅管理**：管理已订阅的角色列表

#### 6. 积分系统
- **每日签到**：每日签到获得积分奖励
- **积分查看**：实时查看当前积分余额
- **兑换码**：使用兑换码兑换积分或其他奖励

#### 7. 评分与评论
- **角色评分**：对AI角色进行1-5星评分
- **角色评论**：发表对角色的评论和反馈
- **评分统计**：查看角色的平均评分和评论列表

### 管理员端功能

#### 1. 用户管理
- **用户列表**：查看所有注册用户
- **用户状态管理**：启用/禁用用户账号
- **用户信息查看**：查看用户详细信息、积分、创建时间等

#### 2. 角色管理
- **角色审核**：审核用户提交的角色
- **AI自动审核**：ai自动审核用户提交的角色
- **角色管理**：编辑、启用/禁用角色
- **角色统计**：查看角色使用次数、评分等数据

#### 3. AI模型管理
- **模型配置**：添加和管理多个AI模型（如GPT-3.5、GPT-4等）
- **API配置**：配置模型的API地址和密钥
- **参数设置**：设置温度、最大token数等模型参数
- **模型排序**：设置模型的显示顺序

#### 4. 分类管理
- **分类创建**：创建和管理角色分类
- **分类排序**：设置分类的显示顺序

#### 5. 系统设置
- **网站配置**：设置网站标题、描述、Logo等基本信息
- **功能开关**：控制注册功能、签到功能等
- **积分设置**：配置签到奖励积分数量
- **公告管理**：发布和管理系统公告
- **邮件配置**：配置SMTP邮件服务器，用于发送验证码

#### 6. 兑换码管理
- **生成兑换码**：批量生成兑换码
- **兑换码管理**：查看兑换码使用情况、有效期等

#### 7. 管理员管理
- **管理员列表**：管理后台管理员账号
- **权限分级**：支持超级管理员和普通管理员两种级别

#### 8. 数据统计
- **用户统计**：总用户数、今日新增用户
- **角色统计**：角色总数、使用情况
- **对话统计**：对话总数、活跃度等

## 技术特点

### 前端技术
- **响应式设计**：支持PC端和移动端访问
- **现代化UI**：采用电影级黑白主题设计，视觉效果出色
- **实时交互**：支持流式数据传输，提供流畅的用户体验
- **Markdown支持**：AI回复支持Markdown格式渲染

### 后端技术
- **PHP + MySQL**：基于PHP和MySQL开发
- **PDO数据库操作**：使用PDO进行安全的数据库操作
- **Session管理**：完善的用户会话管理
- **流式传输**：支持Server-Sent Events (SSE) 实现AI回复的流式输出
- **邮件服务**：集成PHPMailer实现邮件发送功能

### 安全特性
- **密码加密**：使用MD5加密存储密码
- **SQL注入防护**：使用PDO预处理语句防止SQL注入
- **XSS防护**：使用htmlspecialchars防止XSS攻击
- **会话验证**：完善的登录状态验证机制
- **权限控制**：管理员权限分级管理

## 系统要求

### 服务器环境
- **PHP版本**：PHP 7.0 或更高版本
- **数据库**：MySQL 5.5 或更高版本（推荐MySQL 5.7+）
- **Web服务器**：Apache 或 Nginx
- **PHP扩展**：
  - PDO
  - PDO_MySQL
  - cURL（用于API调用）
  - mbstring（用于字符串处理）

### 可选组件
- **Composer**：用于管理PHPMailer依赖（已包含vendor目录）

## 安装说明

### 1. 上传文件
将项目文件上传到Web服务器目录

### 2. 配置数据库
1. 创建MySQL数据库
2. 访问 `http://域名/install/` 进入安装向导
3. 按照提示填写数据库配置信息
4. 系统会自动创建数据库表结构

### 3. 配置邮件服务（可选）
如果需要使用邮件验证功能，需要在后台配置SMTP服务器信息

### 4. 配置AI模型
在管理员后台添加AI模型配置，包括：
- API地址
- API密钥
- 模型参数（温度、最大token等）

## 使用说明

### 用户使用流程

1. **注册账号**
   - 访问注册页面
   - 填写用户名、邮箱、密码
   - 输入邮箱验证码完成注册

2. **浏览角色**
   - 在首页或角色广场浏览AI角色
   - 使用搜索和筛选功能找到心仪的角色
   - 查看角色详情和评分

3. **开始对话**
   - 点击角色卡片进入角色介绍页
   - 点击"开始聊天"进入对话页面
   - 输入消息与AI角色对话

4. **创建角色**
   - 进入"我的角色"页面
   - 点击"创建角色"
   - 填写角色信息并保存

5. **每日签到**
   - 在用户主页点击"立即签到"
   - 每日首次签到可获得积分奖励

### 管理员使用流程

1. **登录后台**
   - 访问 `/admin/login.php`
   - 使用管理员账号登录

2. **配置系统**
   - 在"系统设置"中配置网站基本信息
   - 配置邮件服务器（如需要）
   - 设置签到积分等参数

3. **添加AI模型**
   - 在"AI模型管理"中添加模型
   - 配置API地址和密钥
   - 设置模型参数

4. **管理角色**
   - 审核用户提交的角色
   - 管理公开角色的状态
   - 查看角色使用统计

5. **管理用户**
   - 查看用户列表
   - 管理用户状态
   - 查看用户数据

## 注意事项

1. **数据库配置**：首次使用需要通过安装向导配置数据库
2. **文件权限**：确保 `logs/` 目录有写入权限
3. **API配置**：需要配置有效的AI模型API才能正常使用聊天功能
4. **邮件配置**：如需使用邮件验证功能，需要配置正确的SMTP服务器
5. **安全建议**：生产环境建议关闭错误显示，修改默认管理员密码
